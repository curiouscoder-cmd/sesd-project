# Friends Expense Splitter

A web app to track and split shared expenses among friends. Create groups, add expenses, choose how to split them, and see who owes whom.

## Tech Stack

- **Framework:** Next.js (TypeScript)
- **Backend:** Next.js API Routes
- **ORM:** Prisma
- **Database:** MySQL
- **Auth:** JWT
